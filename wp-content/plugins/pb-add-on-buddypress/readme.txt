== Description ==

Profile Builder - BuddyPress Integration Add-on

Integrates Profile Builder with BuddyPress, allowing you to transfer BuddyPress user profile to Profile Builder user profile automatically.

Replaces BuddyPress Registration, Profile Editing, Profile Viewing and Member Listing with the extended user profiles created with Profile Builder.
You can choose to import existing BuddyPress fields and user entries to Profile Builder.


NOTE:

In order to function properly this plugin requires that you have both BuddyPress and Profile Builder plugins installed

== Installation ==

1. Upload and install the zip file via the built in WordPress plugin installer.
2. Activate the plugin from the "Plugins" admin panel using the "Activate" link.
3. Go to BuddyPress tab under Profile Builder menu and you'll have access to BuddyPress integration settings.
